Create your task sheet and give it a title

Choose Tasks editor from the top menu to start editing your new tasks sheet. Click + Add a title to pull down a title edit box. To start entering the title click on a placeholder text in the center of the box. To delete an existing title click Delete button in the top right corner of the title edit box. Don't worry if you accidentally delete the title — its text will be restored if you click + Add a title button again. And, by the way, the title is optional — don't make it up if you don't need it.

Add a few tasks

Click a big green + Add a new task button — task editor will open (note that you can close it at any time abandoning any changes you've made by clicking Cancel button in its top right corner). The editor consists of three blocks — two operands and an operation between them. Operand blocks are identical, they allow you either to select a number of objects to display (3x3 icons with dots, from one to nine) or to enter a number (bottom row). Operation block allows you to select either one of arithmetic operations or a lower-tnan/greater-than gizmo.

Once you've selected values in all three blocks, Save button appears in the top right corner of the task editor — click it to save your changes. Now you can either Edit or Delete the task by pressing a corresponding button in the top right corner of the task row (note that these buttons only appear when you hover the row with a mouse).

You can add as many tasks as you like to your sheet, but note that when you print it out, a standard A4 page will fit 5 tasks in portrait orientation and 10 in landscape.

Print them out

After you've added at least one task to your sheet Print out the sheet below button appears in the top menu on the page — click it to invoke your browser's printer dialog. Please note that by changing page orientation you can control physical size of printed tasks and, as a result, how many of them fit on a single page. In portrait orientation tasks will be quite big and there will be 5 of them per A4 page — that suits smaller children. In landscape orientation tasks become smaller and 10 of them will fit on a page.

Save or share a task sheet

You may have noticed by this point that all the changes you apply to the sheet synchronously change a URL of the web page in an address bar of your browser. The reverse is also true — if you figure out the format (and it is not a rocket science) you will be able to change your task sheet by changing the URL of the page. Anyway, if you need to save a task sheet for later — just copy the URL of the page from your browser's address bar into clipboard and paste it into a text editor of your choice. If you want to share the sheet with someone — just email them that URL.
